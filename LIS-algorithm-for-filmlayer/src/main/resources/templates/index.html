FILM 레이어 증분편집 — LIS Diff PoC

편집 전(before) / 후(after) 스냅샷을 비교해 삭제·생성·이동·수정 최소 연산을 계산합니다. 이동 최소화에 LIS(최장 증가 부분수열)를 씁니다 — 한 장이 빠지고 끼어 밀린 것은 이동으로 세지 않습니다.
레이어 번호 규약: 0 = Baseline(맨 아래), 위로 갈수록 증가.

편집 전 (before) — base 스냅샷

#Material NameThickRough

#Material NameThickRough
행을 드래그하면 순서가 바뀝니다. ✕로 삭제, 입력창에서 값 수정.

이동 최소성 — 공통 개 중 앵커 |LIS| = → 이동 개 (= 공통 − 앵커, 수학적 최소)

① after 스택 분석 — 스택 그림과 동일하게 맨 아래가 layer 0. 공통 레이어는 before 위치번호로 치환해 LIS(최장 증가 길이 L)를 태운다. LIS에 든 것이 앵커(유지), 벗어난 것이 이동, before가 없으면 생성.
